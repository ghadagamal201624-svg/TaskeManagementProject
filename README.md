# TaskeManagementProject
Capstone project 
## Description
A comprehensive task management system built with Django. The project is organized into modular applications to handle different planning scopes.

## Features
- **Todo Lists**: Create and manage multiple task groups.
- **Tasks**: Specific items within lists with priority levels and due dates.
- **Weekly & Monthly Plans**: Specialized apps for long-term planning.

## Installation
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run migrations: `python manage.py migrate`.
4. Start the server: `python manage.py runserver`.

## Technologies Used
- Python / Django
- SQLite (Development Database)
